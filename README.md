# SQL Lab

An interactive SQL practice platform where your queries run against a real SQL engine in the browser.

**Live site:** [sql-labs.lovable.app](https://sql-labs.lovable.app/)

## Modes

### Progressive Mode
Work through 20 structured questions that ramp from SELECT basics to complex JOIN + HAVING queries. Each question shows the table schema inline, includes difficulty and topic tags (Beginner, Intermediate, Advanced), and provides hints and reference material below the editor.

### Time Attack
Race the clock: solve as many SQL questions as you can in 5 minutes. Harder questions earn more points (Beginner = 1pt, Intermediate = 2pt, Advanced = 3pt) and a streak of 3+ correct answers activates a 2x bonus multiplier.

### Visual Guide
Learn SQL concepts through visual examples covering joins, grouping, filtering, and more.

## Features

- In-browser SQL editor with syntax highlighting
- Live query execution against a real SQL engine
- Inline table previews with sample data
- Difficulty tagging and progress tracking
- Hints and concept explanations alongside each question
